# Barbershop Landing Page

Uma landing page moderna e responsiva para uma barbearia — projeto pessoal criado por Thiago Araújo (thiagoarauj01) para praticar e demonstrar habilidades em desenvolvimento front-end.

Resumo
------
Esta landing page apresenta a identidade visual da barbearia, descreve serviços oferecidos, exibe uma galeria de cortes e facilita contato/agendamento. Foi construída com foco em usabilidade, desempenho e responsividade — ideal para portfólio.

Principais recursos
-------------------
- Layout mobile-first e totalmente responsivo
- Menu hambúrguer para dispositivos móveis

- Microinterações e transições suaves
- Validação básica do formulário de contato
- Otimizações básicas de imagem e performance

Como eu construí (passo a passo)
-------------------------------
1. Planejamento
   - Definição das seções essenciais para conversão e experiência do usuário.
   - Wireframes iniciais (papel ou Figma) para fluxo visual e hierarquia.
2. HTML semântico
   - Uso de tags como `<header>`, `<main>`, `<section>` e `<footer>` para SEO e acessibilidade.
3. CSS (mobile-first)
   - Estilos base para dispositivos móveis e media queries para telas maiores.
   - Layouts com Flexbox para alinhamentos e Grid para galerias.
   - Variáveis CSS para facilitar troca de paleta.
4. JavaScript
   - Toggle do menu hambúrguer.
   - Validação simples do formulário com feedback visual.
   - (Opcional) Slider de depoimentos/galeria.
5. Otimização e testes
   - Compressão de imagens e verificação de performance.
   - Testes em diferentes tamanhos de tela e navegadores.

O que eu aprendi
----------------
- Aplicar design responsivo de forma prática (mobile-first, breakpoints).
- Combinar Flexbox e Grid para layouts flexíveis e consistentes.
- Importância do HTML semântico para SEO e acessibilidade.
- Técnicas básicas de performance (compressão de imagens, redução de requests).
- Boas práticas de fluxo de trabalho com Git e deploy simples.

Como rodar localmente
---------------------
1. Clone o repositório:
   git clone https://github.com/thiagoarauj01/barbershop-landing-page.git
2. Entre na pasta:
   cd barbershop-landing-page
3. Abra `index.html` no navegador ou rode um servidor local:
   - Python 3: `python -m http.server 8000`
   - Acesse: http://localhost:8000
