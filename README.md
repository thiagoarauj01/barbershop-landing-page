# Barbershop Landing Page

Uma landing page moderna e responsiva para uma barbearia — projeto pessoal criado por Thiago Araújo (thiagoarauj01) para praticar e demonstrar habilidades em desenvolvimento front-end.

Resumo
-------
Este repositório contém os arquivos de uma landing page para uma barbearia, construída com foco em design limpo, usabilidade e performance. A página apresenta seções como hero, serviços, galeria, depoimentos e contato/agendamento — tudo pensado para converter visitantes em clientes.

Principais recursos
-------------------
- Layout responsivo (mobile-first)
- Navegação com menu hambúrguer em dispositivos móveis
- Seção de serviços, galeria de cortes e formulário de contato
- Microinterações e transições suaves
- Otimizações de imagem e desempenho

Tecnologias
-----------
- HTML5 semântico
- CSS3 (Flexbox, Grid, variáveis CSS)
- JavaScript (interações e validações)

Como eu construí (passo a passo)
--------------------------------
1. Planejamento e wireframes
   - Estruturei as seções essenciais da landing page com foco em conversão e clareza.
2. Estrutura HTML
   - Usei tags semânticas para melhorar SEO e acessibilidade: `<header>`, `<main>`, `<section>`, `<footer>`.
3. Estilização
   - Abordagem mobile-first: criei estilos base para telas pequenas e apliquei media queries para telas maiores.
   - Combinei Flexbox e Grid para diferentes áreas (nav, layout principal, galeria).
   - Adicionei variáveis CSS para facilitar o gerenciamento da paleta de cores.
4. Interatividade
   - Menu responsivo com toggle via JavaScript.
   - Validação básica do formulário e feedback visual ao usuário.
5. Testes e otimização
   - Compressão de imagens, verificação de performance e testes em diferentes tamanhos de tela.

O que aprendi
-------------
- Como aplicar princípios de design responsivo e combinar Flexbox com Grid.
- Boas práticas de HTML semântico e acessibilidade (uso de `alt`, estrutura de headings, foco visível).
- Técnicas básicas de otimização de performance para sites estáticos.
- Como planejar e transformar um design em código funcional.

Como rodar localmente
---------------------
1. Clone o repositório:
   git clone https://github.com/thiagoarauj01/barbershop-landing-page.git
2. Entre na pasta do projeto:
   cd barbershop-landing-page
3. Abra `index.html` no navegador ou rode um servidor local básico:
   - Com Python 3: `python -m http.server 8000` e acesse `http://localhost:8000`
