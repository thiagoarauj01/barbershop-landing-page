Como eu construí (passo a passo)
-------------------------------
1. Planejamento
   - Definição das seções essenciais para conversão e experiência do usuário.
   - Wireframes iniciais (papel ou Figma) para fluxo visual e hierarquia.
2. HTML semântico
   - Uso de tags como `<header>`, `<main>`, `<section>` e `<footer>` para SEO e acessibilidade.
3. CSS (mobile-first)
   - Estilos base para dispositivos móveis e media queries para telas maiores.
   - Layouts com Flexbox para alinhamentos e Grid para galerias.
   - Variáveis CSS para facilitar troca de paleta.
4. JavaScript
   - Toggle do menu hambúrguer.
   - Validação simples do formulário com feedback visual.

   - Compressão de imagens e verificação de performance.
   - Testes em diferentes tamanhos de tela e navegadores.
